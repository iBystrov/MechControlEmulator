Тестовое задание.

Необходимо разработать небольшое консольное приложение на языке C++ 
с использованием Qt (без QML) для имитации работы системы управления
механизмом, который задает положение объекта в пространстве. 

Описание механизма:

Механизм состоит из двух осей, каждая из которых управляется отдельным
шаговым двигателем. 
•	Ось X - вращение в горизонтальной плоскости.
•	Ось Y - вращение в вертикальной плоскости.

Приложение должно:

1.	Принимать команды от пользователя:
     `move x <значение>` - переместить объект по оси X на заданное значение.
     `move y <значение>` - переместить объект по оси Y на заданное значение.
     `get position` - вывести текущее положение объекта по осям X и Y.
     `exit` - завершить работу приложения.

2.	Имитировать движение:
     При получении команды `move`, приложение должно вывести в консоль 
     информацию о текущем положении объекта и о том, на сколько шагов 
     было перемещено устройство по каждой оси.

3.	Хранить данные:
     Приложение должно сохранять текущее положение объекта в текстовый файл. 

Требования:

•	Приложение должно быть разработано на C++ с использованием Qt.
•	Использование QML не допускается.
•	Код приложения должен быть структурирован, легко читаемым и комментированным.
•	Приложение должно быть работоспособным и соответствовать описанию.

Дополнительные рекомендации:

•	Используйте библиотеку Qt для обработки ввода пользователя и вывода информации в консоль.
•	Для имитации движения можно использовать простой алгоритм, который увеличивает/уменьшает 
значение координаты на заданное количество шагов.
•	Файл с данными можно сохранить в формате CSV.
