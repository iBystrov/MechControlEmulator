---- Инструменты ----
Qt версии 6.7.2
IDE Visual Studio 2022 с расширением Qt VS Tools 3.2.0
Система сборки: MSBuild
---------------------

---- Примечания ----
1. Перемещения по осям X, Y ограничены значениями ±20000.
(Задаются константами в классе ControlSystem).

2. Значения в консоль вводить без угловых скобок.
Примеры:
	move x 555
	move y -777

3. CSV-файл следует просматривать в текстовом редакторе, например Notepad++.
(При открытии в Excel не отображаются нулевые значения и запятые рядом с ними).

4. Кодировка исходников UTF-8.
